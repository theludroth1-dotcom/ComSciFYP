# ComSciFYP
The functional and component test of the Computer Science Local Disaster Preparedness App Final Report

To run this code in Visual Studio Code
1. Run in command prompt npm install jest-expo jes
2. Run in command prompt npm install @testing-library/react-native

3.In package.json add these following config
"scripts": {
  "test": "jest"
},
"jest": {
  "preset": "jest-expo/universal"
}

4.Run in command prompt npm install react@19.2.4 react-test-renderer@19.2.4 --save-exact
5.Run in command prompt npm test
